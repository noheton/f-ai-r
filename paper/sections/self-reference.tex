\section{This Paper, Generated by this Process}
\label{sec:self}

The fifth integrated practice (\S\ref{sec:eight}, item~5) requires
that a paper which prescribes a discipline have been written under
that discipline. This section is the explicit attestation; every
claim below points to an artefact in the public repository.

\paragraph{What ran.}
The ten-stage pipeline of \S\ref{sec:pipeline} is materialised as
one prompt file per role under \texttt{agents/}. The harness was
Anthropic Claude (\texttt{claude-opus-4-7}) operating under the
rules in \texttt{CLAUDE.md}. The Orchestrator routed work; the
Scientific Writer drafted prose; the Source Analyzer promoted
retrieved sources; the scrutinizers filed defect registries; the
Provenance Curator alone wrote \texttt{doc/provenance.ttl}; the Site
Agent rendered the public site.

\paragraph{What is committed.}
The manuscript (\texttt{paper/main.tex} plus
\texttt{paper/sections/*.tex}); the agent prompts
(\texttt{agents/}); the provenance graph
(\texttt{doc/provenance.ttl}, valid Turtle, parsed by \texttt{rdflib}
in CI); the logbook (\texttt{doc/logbook.md}, append-only, ISO-8601
dated); the build pipeline (\texttt{.github/workflows/}). Every named
claim in this manuscript has a corresponding \texttt{fair2r:Claim}
IRI in the graph.

\paragraph{Replay.}
\texttt{git clone}; install TeX Live and the Python dependencies;
\texttt{make -C paper pdf};
\texttt{python scripts/build\_provenance\_site.py}. Compare against
the published artefacts; any difference is a defect.

\paragraph{Honest limits.}
The attestation is artefactual, not behavioural. It demonstrates
that the eight practices and the ten-stage pipeline produced an
artefact set that looks the way the prescription says it should
look. It does not, by itself, demonstrate that the human author
resisted every shortcut the pattern was meant to prevent --- that
depends on review. \emph{If the graph and the prose disagree, the
graph wins; that is a defect, and the repository owes a fix for
it.}
